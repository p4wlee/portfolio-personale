## Struttura dei file CSS/SCSS
- **scss/**
  - `_variables.scss` - Definizione delle variabili globali.
  - `_buttons.scss` - Stile dei pulsanti.
  - `style.scss` - File principale che importa tutti i partials.
  - **partials/**
    - `_body.scss` - Stile globale del body.
    - `_header.scss` - Stile dell'header e navbar.
    - `_header2.scss` - Stile aggiuntivo per header secondari.
    - `_home.scss` - Stile specifico per la pagina home.
    - `_footer.scss` - Stile del footer.
    - `_contatti.scss` - Stile per la pagina contatti.
    - `_cv.scss` - Stile per la pagina CV.
